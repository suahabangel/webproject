
Bio-FormulaGPT 사용법 안내서

1. 수식 검색 및 해설
- 메인 페이지에서 수식을 선택하면 MathJax로 시각화됨
- "해설 보기" 버튼 클릭 시 GPT가 한국어로 설명 제공

2. 연습문제
- 각 수식 하단의 "문제 생성" 버튼 클릭 시 GPT 기반 객관식 문제 출력

3. 챗봇 사용법
- 질문 입력창에 자유롭게 과학적 질문 입력
- GPT 튜터가 실시간으로 답변 제공

4. 외부 연동
- PubChem → 화합물명 입력 시 자동 조회
- NCBI → 유전자 이름으로 검색 가능

5. 배포 환경
- Vercel: 프론트엔드
- Render: NestJS 백엔드
- Hugging Face: Streamlit 기반 챗봇 데모
